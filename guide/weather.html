    # 天气查询

    使用高德API完成改命令，支持全国所有地区天气实时 和 [今天，明天，后天，大后天] 天气查询，并可设置系统地区。

    $ asd tq [options] [code]

    或

    $ asd weather [options] [code]

    # options参数

    name alias desc
    -b --base 查询实时天气
    -a --all 默认值:查询今天，明天，后天，大后天 天气
    -fc --findcode 查询全国地区code码
    -ss --setsystem 通过code码设置系统地区

    code 为高德天气地区 城市编码
